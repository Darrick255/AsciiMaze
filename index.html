Asciimaze

A simple maze game built with python.

View project on GitHub

ASCII Maze is a work in progress

This means that it is incomplete and may be missing the features discussed here. The development is done on my free time after work and school.

What and Why?

I have been using Python to do small task and compete in coding challenges, I Decided I would make my own simple game to learn python a bit better and to learn git. I Chose a Maze because it is simple enough but with room for complexity. As of publishing this page the maze contains the following features:

  • Recursively and iterativly generate a maze using backtracking.
  • Draw The Maze With ASCII characters
  • Draw the Maze better With different character set. This takes up more space but looks nicer and gives room for items in the maze.
  • Draw the maze using Turtle graphics. this makes everything look polished.
  • Solve the maze from an arbitrary point(player) using a* and Manhattan distance heuristic.
    • Draw the path on the maze using <>^V (only works the second maze drawing method above).

What to do now

I slowly want to expand this into a full functioning enjoyable game. the features that would be needed are:

  • A (very) simple game loop that will control the flow of the game.
  • Controls for the player
    • WASD to move
    • H to get a hint of which way to move (will have a limit)
    • Esc will give a menu of quitting/putting the player on auto pilot and watching the sprint to the finish.
  • pickups
    • food: lets you use the hints more often
    • roll of string: lets you see the tiles you have walked on before.
    • Glasses: enlarge your view of the maze.
  • timer so you can have a top score
  • Save/load Seed/Game to continue or replay a certain maze.
  • Unlikely but a multiplayer race would be a fun addition.
